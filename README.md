# Northwind Data Analysis: Connecting SQL with Python

## 📌 Project Overview
هذا المشروع يقدم تحليلًا شاملاً لقاعدة بيانات **Northwind** الشهيرة، حيث تم الربط بين **SQL Server** و **Python** لاستخراج البيانات وتحليلها. يهدف المشروع إلى الإجابة على أسئلة تجارية مهمة تتعلق بالمبيعات، العملاء، والمنتجات.

## 🛠️ Tech Stack
* **Database:** SQL Server (Northwind Database).
* **Language:** Python.
* **Libraries:** * `pyodbc` أو `sqlalchemy` (للاتصال بقاعدة البيانات).
    * `pandas` (لمعالجة البيانات).
    * `matplotlib` & `seaborn` (للتصور البياني).

## 🚀 Key Features & Analysis
في هذا المشروع، قمت بالآتي:
1.  **Database Connection:** إنشاء اتصال آمن بين Python وبيئة SQL Server.
2.  **SQL Querying:** كتابة استعلامات معقدة لاستخراج بيانات الطلبات (Orders)، الموظفين (Employees)، والموردين (Suppliers).
3.  **Data Cleaning:** معالجة القيم المفقودة وتحويل أنواع البيانات داخل Python.
4.  **Exploratory Data Analysis (EDA):**
    * تحليل أكثر المنتجات مبيعاً.
    * تحديد العملاء الأكثر ربحية.
    * تحليل أداء الموظفين في المبيعات.
    * تتبع نمو الإيرادات الشهري.
